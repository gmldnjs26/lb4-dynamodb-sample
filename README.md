## Setup

```sh
git clone --recurse-submodules https://github.com/gmldnjs26/lb4-dynamodb-sample.git

or

git clone https://github.com/gmldnjs26/lb4-dynamodb-sample.git
git submodule update --init --recursive
```

```sh
npm install
```

```sh
docker-compose up
```
